# NBA Win Prediction Mini Project

## Project Title
NBA Win Prediction Mini Project

## Description
This project aims to predict the outcomes of NBA games using historical data and machine learning techniques.

## Goal Overview
The goal of this project is to develop a predictive model that can determine the winner of an NBA game based on various historical and statistical factors.

## Data Source
The data for this project is sourced from the NBA API, which provides access to a comprehensive dataset regarding games, player statistics, and more.

## Setup Instructions
To get started with this project, follow these steps:
1. Clone this repository: `git clone https://github.com/JakeRoll04/nba-win-prediction-mini-project.git`
2. Navigate to the project directory: `cd nba-win-prediction-mini-project`
3. Install the required dependencies using pip:
   ```
   pip install -r requirements.txt
   ```
4. Open the Jupyter Notebook and run the cells to execute the project.

## Workflow Description
The workflow of this project consists of several phases:
1. **Data Collection:** Data is fetched from the NBA API for the required seasons.
2. **Data Preprocessing:** The data is cleaned and prepared for modeling, including handling missing values and feature selection.
3. **Modeling:** Various machine learning models are employed to predict game outcomes.
4. **Evaluation:** The models are evaluated based on their accuracy and other performance metrics.
