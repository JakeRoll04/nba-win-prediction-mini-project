# NBA Win Prediction Mini Project

This repository contains a mini project for predicting NBA game outcomes using machine learning techniques. 

## Project Structure
- `data/`: Contains datasets used for training and testing the model.
- `notebooks/`: Contains Jupyter notebooks for various analyses and model training.
- `src/`: Contains source code, including model training and prediction scripts.
- `tests/`: Contains unit tests for the source code.

## Setup
To set up the project, install the required packages listed in `requirements.txt`. Run the following command:
```bash
pip install -r requirements.txt
```
